Ce projet vise à créer un site interactif permettant d'enregistrer dans une base de données les informations saisies par les visiteurs dans un formulaire HTML. Dans le contexte d'un site de e-commerce, les fonctionnalités incluent la saisie et l'insertion des coordonnées d'un client dans la table "client" de la base de données "magasin", la mise à jour de ces informations, et une fonction de recherche selon des critères spécifiques. La base de données "Magazin" est créée avec PhpMyAdmin, comprenant les tables "client", "commande" et "ligne". Le développement se fait en utilisant PHP 7 et PDO pour la manipulation des données.
